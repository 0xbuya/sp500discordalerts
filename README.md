# S&P 500 discord alerts bot
Free Discord Bot for S&P 500 Insider Trading Alerts tracking real-time SEC Form 4 filings (insider buys/sells) for S&amp;P 500 companies using a configurable API (currently utilizing Finnhub). Clean net activity + recent buys/sells with insider names, shares, prices, and ownership Saves raw CSV for analysis Optional Twitter posts


Features:
On-demand summaries via !insider [days] command
Clean net activity + recent buys/sells with insider names, shares, prices, and ownership
Saves raw CSV for analysis
Optional Twitter posts
